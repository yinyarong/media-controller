Privacy Policy

Last updated: May 7, 2026

Overview

Media Controller is a Chrome extension that lets you control the playback speed and loop behavior of video and audio elements on any webpage. This policy explains how the extension handles information.

Data Collection

Media Controller does not collect, store, or transmit any personal data. The extension does not use analytics, crash reporting, or any third-party services. No information ever leaves your device.

Permissions

The extension requests the following permissions solely to provide its core functionality:

  • activeTab — allows the popup to communicate with the currently active tab to read and control media elements on that page.
  • scripting — required by Chrome to inject the content script that detects and manages media elements.
  • Host access (<all_urls>) — the content script runs on all pages so it can detect media elements regardless of which site you visit. No page content is read or stored.

Local Storage

Media Controller does not use chrome.storage, localStorage, IndexedDB, or cookies. No data is persisted between sessions.

Third Parties

The extension does not include any third-party code, advertising SDKs, or tracking libraries. All code runs locally on your device.
